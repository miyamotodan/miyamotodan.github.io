Manuale d'Uso

ImgMeasure è uno strumento per misurare distanze su immagini direttamente nel browser, senza installazioni. Ideale per rilevare quote da fotografie, scansioni e disegni tecnici.

1. Carica un'immagine
Clicca su Carica e seleziona un file immagine. Le misure già esistenti vengono mantenute — usa per eliminarle se vuoi ripartire da zero.
2. Modalità di disegno

Scegli la modalità dalla barra degli strumenti, poi clicca (o tocca) sul canvas per disegnare.

Selezione — seleziona e trascina i vertici per riposizionarli
Segmento — due click per una linea di misura singola
Continuativo — ogni punto finale diventa l'inizio del successivo; ideale per polilinee
Rettangolo — due click su vertici opposti creano 4 segmenti con larghezza e altezza
Cerchio — primo click sul centro, secondo sul perimetro; misura raggio e diametro
3. Scala di riferimento
  1. Disegna un segmento su una dimensione nota nell'immagine
  2. Cliccaci sopra per selezionarlo — la lunghezza in pixel si compila automaticamente in Impostazioni
  3. Inserisci la misura reale corrispondente in millimetri e clicca Applica rapporto
  4. Tutte le misure si aggiornano in tempo reale
4. Zoom e navigazione
Desktop Tasto destro + trascina per spostare • / per zoom • per reset vista • doppio click per reset zoom e annullamento disegno in corso
Touch / Pennino Pinch per zoom • un dito per spostare • doppio tap per reset zoom e annullamento disegno in corso
5. Aggancio automatico (snap)
Quando il cursore o la penna si avvicina a un vertice esistente (entro 20 px schermo), compare un indicatore circolare e il punto si aggancia automaticamente — indipendentemente dal livello di zoom. Funziona sia durante il disegno sia quando si trascinano vertici in modalità Selezione.
6. Salvataggio automatico
Le misure vengono salvate automaticamente nel browser ad ogni modifica. Al prossimo caricamento di una immagine, i segmenti vengono ripristinati. Per cancellare tutto usa .
7. Controlli aggiuntivi
Attiva/disattiva l'anteprima in tempo reale durante il disegno
Mostra/nasconde tutti i segmenti (utile per confrontare con l'immagine originale)
Annulla l'ultimo segmento creato (anche Ctrl+Z)
Annulla il disegno in corso senza creare il segmento (anche ESC)
Cambia tema ciclicamente: Scuro → Chiaro → Alto Contrasto
User Manual

ImgMeasure is a browser-based tool for measuring distances on images — no installation required. Ideal for extracting dimensions from photos, scans and technical drawings.

1. Load an image
Click Upload and select an image file. Existing measurements are kept — use to delete them if you want to start over.
2. Drawing modes

Select a mode from the toolbar, then click (or tap) on the canvas to draw.

Select — select and drag vertices to reposition them
Segment — two clicks to create a single measurement line
Continuous — each end point becomes the start of the next; ideal for polylines
Rectangle — two clicks on opposite corners create 4 segments with width and height
Circle — first click on center, second on perimeter; measures radius and diameter
3. Reference scale
  1. Draw a segment over a known dimension in the image
  2. Click it to select it — its pixel length fills automatically in Settings
  3. Enter the corresponding real-world measurement in millimeters and click Apply ratio
  4. All measurements update in real time
4. Zoom and navigation
Desktop Right-click + drag to pan • / to zoom • to reset view • double-click to reset zoom and cancel current drawing
Touch / Stylus Pinch to zoom • one finger to pan • double tap to reset zoom and cancel current drawing
5. Automatic snapping
When the cursor or stylus approaches an existing vertex (within 20 screen pixels), a circular indicator appears and the point snaps automatically — regardless of zoom level. Works both during drawing and when dragging vertices in Select mode.
6. Auto-save
Measurements are saved automatically in the browser after every change. The next time you load an image, segments are restored. Use to clear everything.
7. Additional controls
Enable/disable real-time preview while drawing
Show/hide all segments (useful to compare against the original image)
Undo the last created segment (also Ctrl+Z)
Cancel the current drawing without creating a segment (also ESC)
Cycle theme: Dark → Light → High Contrast
